Polymorphism

In this tutorial we will explore the concept of polymorphism.
We have a animal class to represent an animal. The animal class has a speak method.

After we have defined the animal parent class, we’ll define a few child classes using the < operator.
Cow, sheep, and pig are all child classes of the animal class.

Now that we have defined these classes, we will make an array of animals called farm.
Our newly created farm will have a cow, sheep and pig.

Finally the last few lines loop through each element of the farm array.
We ask each animal of the farm to speak.

Although i in the loop is of the animal class, i will give a unique answer each time it is called to speak.

=begin
Name: Polymorhism.rb
Description: An example of polymorphism in ruby.
The parent class Animal has three child classes, Cow, Sheep, and Pig.
At the end we will ask each element of an array of animals to speak, all will give unique responses.
Author: Kai Boschma
Date: September 30th 2012
=end

#This will be the parent class to the farm animals below
class Animal
 def speak
 print ""
 end
end

#A class for representing a cow
#Note that the operator < designates that Cow is a child class of Animal
class Cow < Animal
 def speak
 puts "Moo"
 end
end

#A class for representing a sheep
class Sheep < Animal
 def speak
 puts "Baa"
 end
end

#A class for representing a pig
class Pig < Animal
 def speak
 puts "Oink"
 end
end

#The array farm, holds 3 animals
#They are a cow, a sheep, and a pig
farm = [Cow.new, Sheep.new, Pig.new]

#Here we go through every animal in farm
#For each animal we tell it to speak
#Running the program will show that each animal will speak, but each will make their own sound
for i in farm
 i.speak()
end
